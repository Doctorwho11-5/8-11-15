This page will teach you about about text and how to change the size and color of text.
First of all, as far as size goes, the numbers are backwards. So if you dont want to change the size, 
like this text, then just leave it. If you want to change it, then type <h> and a number after the "h",
ranging from 1-6, above your text, but under the <p> tag. 
Now size 6 is really small, so use this for small text, size 1 is large text. So use it like this, 


HUGE TEXT <h1>


Large Text <h2>


Big Text <h3>


Medium Text <h4>


Small Text <h5>


Tiny Text <h6>
 

Now for bold text, use <b> and close it off with </b>
for Italics, use <i> and close it off with </i>

Saving and opening your page to see yor texts and colors is simple. First go up to "File" in the upper left, 
Then click "Save as". Now type the name of your page and put ".html" at the end. For example, if I want to call 
my page burgers, then I would type "burgers.html", now click save. Now close the notepad and open the link you just made! 
Congradulations! You now know the basics of page making.
Now if you want to edit it again, open the page through the link, right click and click "veiw page source", then copy and paste
the code into a notepad(Text Document) and edit. Now to save it to your page, click "save as" and type in your name
again, you should now be able to edit it forever if you edit and click "save".
